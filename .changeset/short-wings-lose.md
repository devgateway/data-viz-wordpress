This the first release of @devgateway/create-wp-customizer, a command-line tool for quickly scaffolding Data Viz WordPress Customizer projects.
Features
- Interactive CLI for project creation
- Two starter templates:
    - JavaScript template (template-js)
    - TypeScript template (template-ts)
- Simple command-line options for non-interactive usage
- Project scaffolding with proper folder structure
